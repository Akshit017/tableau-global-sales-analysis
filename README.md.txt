# Global Sales Performance & Customer Analytics

## Project Overview
This project analyzes global sales data to uncover insights related to sales performance, profitability, customer behavior, and regional trends using Tableau.

## Dashboards
### 1. Sales Overview
- Tracks total sales, orders, and profit margin
- Monthly sales trends and regional performance

### 2. Profit & Product Analysis
- Identifies loss-making products
- Analyzes discount vs profit relationship
- Highlights top and bottom performing products

### 3. Customer & Segment Analysis
- Repeat vs one-time customer analysis using LOD calculations
- Segment-wise and region-wise customer performance
- Top customer identification

## Tools Used
- Tableau
- Data Visualization
- Business Analysis
